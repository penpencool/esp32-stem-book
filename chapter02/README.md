## 📖 บทนำ: ทำไมต้องติดตั้งโปรแกรม?

### 🔧 เครื่องมือเขียนโค้ดคืออะไร?

ลองนึกภาพว่าเราจะเขียนหนังสือ 📚

- ถ้าเราเขียนด้วย **ดินสอบนกระดาษ** → ก็เขียนได้ แต่ต้องลบแก้ยาก
- ถ้าเราเขียนด้วย **โปรแกรม Word** → มีตัวสะกด มีจัดหน้า สะดวกกว่าเยอะ!
- ถ้าเราใช้ **โปรแกรมเขียนหนังสือเฉพาะทาง** → ยิ่งง่ายขึ้น!

การเขียนโค้ดก็เหมือนกัน!

- ถ้าเราเขียนด้วย **Notepad** (โปรแกรมพื้นฐานของ Windows) → ก็เขียนได้ แต่ไม่มีการช่วยเหลือ
- ถ้าเราเขียนด้วย **VS Code + PlatformIO** → มีการเติมโค้ดอัตโนมัติ ตรวจข้อผิดพลาด และอัปโหลดไปบอร์ดได้เลย!

ดังนั้นการติดตั้งโปรแกรมที่ถูกต้อง = การเตรียมเครื่องมือที่ดีก่อนทำงาน! 🔧

### 💻 VS Code คืออะไร?

**VS Code (Visual Studio Code)** คือ "โปรแกรมแก้ไขโค้ด" ที่ได้รับความนิยมมากที่สุดในโลก พัฒนาโดย Microsoft ฟรีและใช้งานง่าย!

#### ✨ VS Code ทำอะไรได้บ้าง?

- ✍️ **เขียนโค้ด** — รองรับภาษาโปรแกรมมากกว่า 70 ภาษา
- 🎨 **แต่งสีโค้ด** — ทำให้โค้ดอ่านง่าย (Syntax Highlighting)
- 🤖 **เติมโค้ดอัตโนมัติ** — พิมพ์ได้เร็วขึ้น (IntelliSense)
- 🐛 **ตรวจข้อผิดพลาด** — ช่วยหา bug ก่อนรัน
- 📂 **จัดการไฟล์** — เปิดโฟลเดอร์ ไฟล์ได้ง่าย

### 🧩 PlatformIO คืออะไร?

**PlatformIO** คือ "Extension (ส่วนขยาย)" ที่ติดตั้งเพิ่มใน VS Code ทำให้ VS Code สามารถ:

- 🔌 **รองรับบอร์ดหลายตัว** — ESP32, Arduino, STM32 และอื่นๆ
- 📦 **จัดการ Library อัตโนมัติ** — ไม่ต้องติดตั้งเอง
- ⬆️ **อัปโหลดโค้ดง่ายๆ** — กดปุ่มเดียว
- 🔍 **มี Serial Monitor** — ดูข้อความจากบอร์ดได้เลย

## 💻 เนื้อหา: วิธีติดตั้ง VS Code

### 🔹 ดาวน์โหลด VS Code

#### สำหรับ Windows

1. **เปิดเว็บไซต์:** ไปที่ [https://code.visualstudio.com](https://code.visualstudio.com)
2. **กดปุ่ม Download:** จะเห็นปุ่มใหญ่ๆ สีน้ำเงิน "Download for Windows"
3. **เลือก Windows 10/11 (stable):** คลิกแล้วไฟล์จะดาวน์โหลดอัตโนมัติ
4. **เปิดไฟล์ติดตั้ง:** ดับเบิลคลิกที่ไฟล์ `.exe` ที่ดาวน์โหลดมา
5. **ติดตั้ง:** ทำตามขั้นตอน — แนะนำให้เลือก "Add to PATH" ด้วย!

```
📌 ข้อควรระวัง:
- เลือก "User" หรือ "System" installation ก็ได้
- ถ้าเลือก System จะติดตั้งให้ทุกคนในเครื่อง
- ถ้าเลือก User จะติดตั้งเฉพาะคนที่ใช้อยู่
```

#### สำหรับ macOS

1. **เปิดเว็บไซต์:** ไปที่ [https://code.visualstudio.com](https://code.visualstudio.com)
2. **กดปุ่ม Download:** เลือก "Download Mac Universal" หรือ "Download Intel chip" หรือ "Download Apple silicon" ตามเครื่องของเรา
3. **เปิดไฟล์:** ดับเบิลคลิกไฟล์ `.zip` ที่ดาวน์โหลดมา
4. **ย้ายไฟล์:** ลากไฟล์ "Visual Studio Code" ไปไว้ในโฟลเดอร์ Applications
5. **เปิดโปรแกรม:** คลิกที่ไฟล์ใน Applications เพื่อเปิด VS Code

#### สำหรับ Linux (Ubuntu/Debian)

1. **เปิด Terminal:** กด `Ctrl + Alt + T`
2. **ติดตั้งผ่าน Terminal:**

```bash
# อัปเดต package list
sudo apt update

# ติดตั้ง VS Code
sudo apt install code

# หรือดาวน์โหลด .deb file จากเว็บแล้วติดตั้งด้วย:
# sudo dpkg -i <filename.deb>
```

## 🛠️ วิธีติดตั้ง PlatformIO Extension

### 🔹 PlatformIO Extension คืออะไร?

ก่อนหน้านี้เราบอกว่า PlatformIO เป็น "Extension" หรือ "ส่วนขยาย" ของ VS Code คล้ายกับการติดตั้งแอปเพิ่มในมือถือ เช่น เราซื้อมือถือมาแล้ว แต่อยากมี Instagram ก็ต้องติดตั้งเพิ่ม

ในการเขียนโค้ดสำหรับ ESP32-C3 เราต้องติดตั้ง PlatformIO Extension เพิ่มเพื่อให้ VS Code "เข้าใจ" วิธีอัปโหลดโค้ดไปยังบอร์ด

### 🔹 ขั้นตอนการติดตั้ง PlatformIO Extension

#### ขั้นตอนที่ 1: เปิด Extensions Marketplace

1. เปิด VS Code ขึ้นมา
2. คลิกที่ **Activity Bar** ด้านซ้าย (ไอคอนรูปสี่เหลี่ยม 4 ช่อง)
3. คลิก **Extensions** หรือกด `Ctrl + Shift + X`

```
📍 ที่อยู่: ด้านซ้ายสุด → ไอคอนรูปสี่เหลี่ยมจัตุรัส
```

#### ขั้นตอนที่ 2: ค้นหา PlatformIO

1. พิมพ์ **"platformio ide"** ในช่องค้นหา Extensions
2. รอผลการค้นหาสักครู่
3. จะเห็นผลลัพธ์หลายอัน ให้หา **"PlatformIO IDE"** (มีไอคอนรูปเหยี่ยว)

```
ผลการค้นหา:
┌────────────────────────────────────────┐
│ 🔍 platformio ide                     │
├────────────────────────────────────────┤
│ 📦 PlatformIO IDE                     │
│    by PlatformIO                      │
│    The next generation platform for...│
│    [Install] [⭐4.8] [3.3M]            │
│                                        │
│ 📦 PlatformIO for CLion               │
│    by PlatformIO                      │
│    [Install] [⭐4.7] [245K]            │
│                                        │
│ 📦 C/C++ Extension Pack              │
│    by PlatformIO                      │
│    [Install] [⭐4.5] [1.2M]            │
└────────────────────────────────────────┘
```

#### ขั้นตอนที่ 3: ติดตั้ง PlatformIO IDE

1. คลิก **"PlatformIO IDE"**
2. คลิกปุ่ม **"Install"** (สีน้ำเงิน)
3. รอการติดตั้ง... (อาจใช้เวลาประมาณ 5-10 นาที เพราะมีไฟล์ใหญ่)
4. เมื่อติดตั้งเสร็จ ปุ่มจะเปลี่ยนเป็น **"Uninstall"**

#### ขั้นตอนที่ 4: รีสตาร์ท VS Code

1. **ปิด VS Code** แล้วเปิดใหม่
2. หรือกด **"Reload Required"** (ถ้ามีข้อความนี้โผล่มา)

#### ขั้นตอนที่ 5: ตรวจสอบการติดตั้ง

1. ดูที่ **Activity Bar** ด้านซ้าย
2. จะเห็นไอคอน **"Ant" (ตัวต่อ)** เพิ่มขึ้นมาใหม่!
3. ไอคอนนี้คือ **PlatformIO** — คลิกแล้วจะเห็นเมนูของ PlatformIO

> 💡 **เคล็ดลับ:** ถ้าติดตั้งไม่ได้ ลองตรวจสอบว่า VS Code เป็นเวอร์ชันล่าสุดหรือเปล่า ไปที่ Help → Check for Updates

> ⚠️ **ปัญหาที่อาจเจอ:** ถ้าติดตั้งนานมากๆ อาจเป็นเพราะอินเทอร์เน็ตช้า รอได้เลย หรือลองติดตั้งใหม่อีกครั้ง

## 🏗️ สร้าง Project แรก: Hello ESP32

### 🔹 Project คืออะไร?

**Project** คือ "โฟลเดอร์" ที่เก็บไฟล์ทั้งหมดที่เกี่ยวข้องกับงานของเรา เช่น:

```
📁 HelloESP32/          ← โฟลเดอร์ Project
├── 📄 src/
│   └── main.cpp         ← ไฟล์โค้ดหลัก
├── 📄 platformio.ini    ← ไฟล์ตั้งค่า
└── 📄 lib/              ← โฟลเดอร์เก็บ Library
```

### 🔹 ขั้นตอนการสร้าง Project ใหม่

#### ขั้นตอนที่ 1: เปิด PlatformIO

1. คลิกไอคอน **"Ant"** (PlatformIO) ใน Activity Bar ด้านซ้าย
2. รอให้โหลดสักครู่

#### ขั้นตอนที่ 2: สร้าง Project ใหม่

1. คลิก **"+ New Project"** หรือ **"PIO New Project"**
2. จะมีหน้าต่างป๊อปอัปขึ้นมา:

```
┌─────────────────────────────────────────────┐
│ New Project Wizard                          │
├─────────────────────────────────────────────┤
│ Name: HelloESP32                            │
│ Board: ESP32-C3 Dev Module (ESP32C3)        │
│ Framework: Arduino                          │
│ Location: [✅] Use default location         │
│                                             │
│               [Cancel] [Next >]              │
└─────────────────────────────────────────────┘
```

#### ขั้นตอนที่ 3: กรอกข้อมูล Project

1. **ชื่อ Project:** พิมพ์ `HelloESP32`
2. **Board:** พิมพ์ `esp32-c3` ในช่องค้นหา แล้วเลือก **"ESP32C3 Dev Module"**
3. **Framework:** เลือก **"Arduino"**
4. **Location:** เลือก ✅ "Use default location" (จะเก็บใน Documents/PlatformIO/Projects)
5. คลิก **"Finish"** หรือ **"Next >"** แล้ว **"Finish"**

#### ขั้นตอนที่ 4: รอสร้าง Project

1. รอประมาณ 2-5 นาที (ครั้งแรกจะดาวน์โหลด Toolchain และ Framework)
2. ถ้าเป็นครั้งแรก จะมีข้อความ "Installing dependencies..." ให้รอจนเสร็จ
3. เมื่อเสร็จจะเห็นโครงสร้าง Project ใน Explorer ด้านซ้าย

```
📁 HelloESP32
├── 📄 src/
│   └── main.cpp       ← ไฟล์นี้เราจะเขียนโค้ด!
├── 📄 platformio.ini  ← ไฟล์ตั้งค่า
└── 📄 .pio/
```

### 🔹 เขียนโค้ดแรก: Hello ESP32!

ตอนนี้เรามี Project แล้ว มาเขียนโค้ดแรกกัน!

#### ขั้นตอนที่ 1: เปิดไฟล์ main.cpp

1. ใน Explorer ด้านซ้าย คลิกที่ **src/**
2. ดับเบิลคลิกที่ **main.cpp**

#### ขั้นตอนที่ 2: ลบโค้ดเดิมแล้วเขียนโค้ดใหม่

ลบโค้ดที่มีอยู่แล้วแทนที่ด้วยโค้ดนี้:

```cpp
// ============================================
// บทที่ 2: Hello ESP32 - โค้ดแรกของเรา!
// ============================================
// โค้ดนี้จะส่งข้อความ "Hello ESP32-C3!" 
// ไปแสดงบน Serial Monitor

void setup() {
  // เริ่มต้นการสื่อสารกับคอมพิวเตอร์ที่ความเร็ว 115200 bits/วินาที
  Serial.begin(115200);
  
  // รอให้ Serial พร้อม (รอจนคอมพิวเตอร์ตั้งค่าเสร็จ)
  while(!Serial) {
    ; // รอคอมพิวเตอร์เชื่อมต่อ
  }
  
  // พิมพ์ข้อความเริ่มต้น
  Serial.println("========================================");
  Serial.println("   Hello ESP32-C3! สวัสดีจากบอร์ดของเรา!");
  Serial.println("========================================");
  Serial.println("พร้อมรับคำสั่งแล้ว...");
  Serial.println();
}

void loop() {
  // ทำซ้ำไปเรื่อยๆ ทุก 2 วินาที
  Serial.println("บอร์ดกำลังทำงาน... ✅");
  delay(2000);  // รอ 2 วินาที (2000 มิลลิวินาที = 2 วินาที)
}
```

### 🔹 อธิบายโค้ดทีละบรรทัด

```cpp
// ============================================
// บทที่ 2: Hello ESP32 - โค้ดแรกของเรา!
// ============================================
```
📌 **Comment** — ข้อความที่ไม่ใช่โค้ด คอมพิวเตอร์จะข้ามไม่รัน ช่วยให้คนอ่านเข้าใจว่าโค้ดนี้ทำอะไร เริ่มด้วย `//`

---

```cpp
void setup() {
```
📌 **`void setup()`** — ฟังก์ชันที่จะทำงาน **แค่ครั้งเดียว** ตอนเริ่มบอร์ด เหมือนกับการเปิดเครื่องแล้วตั้งค่าเริ่มต้น

---

```cpp
  Serial.begin(115200);
```
📌 **`Serial.begin(115200)`** — สั่งให้บอร์ดเริ่ม "คุย" กับคอมพิวเตอร์ผ่านสาย USB ที่ความเร็ว 115200 bits/วินาที (เรียกว่า "baud rate")

---

```cpp
  while(!Serial) {
    ; // รอคอมพิวเตอร์เชื่อมต่อ
  }
```
📌 **`while(!Serial)`** — รอจนกว่าการเชื่อมต่อ Serial จะพร้อม ถ้ายังไม่พร้อมก็รอต่อไป

---

```cpp
  Serial.println("========================================");
```
📌 **`Serial.println()`** — พิมพ์ข้อความแล้วขึ้นบรรทัดใหม่ ข้อความอยู่ในเครื่องหมาย `"..."`

---

```cpp
void loop() {
```
📌 **`void loop()`** — ฟังก์ชันที่จะทำงาน **ซ้ำไปเรื่อยๆ** ตลอดไป เหมือนกับการทำงานต่อเนื่องของเครื่อง

---

```cpp
  delay(2000);  // รอ 2 วินาที (2000 มิลลิวินาที = 2 วินาที)
```
📌 **`delay(2000)`** — หยุดรอ 2000 มิลลิวินาที (2 วินาที) ก่อนทำอย่างอื่น

## ⬆️ อัปโหลดโค้ดไปยังบอร์ด

### 🔹 ต่อบอร์ด ESP32-C3 กับคอมพิวเตอร์

1. **เสียบสาย USB** เข้ากับบอร์ด ESP32-C3 (ช่อง USB-C หรือ Micro USB ตามรุ่น)
2. **เสียบปลายอีกด้าน** เข้ากับคอมพิวเตอร์
3. **LED บนบอร์ด** ควรติด (แสดงว่าบอร์ดมีไฟ)
4. **Windows อาจติดตั้ง Driver** อัตโนมัติ (รอสักครู่)

> ⚠️ **ข้อควรระวัง:** ถ้าสาย USB ไม่มีสัญญาณข้อมูล (เป็นสายชาร์จอย่างเดียว) จะอัปโหลดไม่ได้! ต้องใช้สายที่มีสายข้อมูลด้วย

### 🔹 อัปโหลดโค้ด

#### ขั้นตอนที่ 1: เลือก Port (COM Port)

1. ดูที่ **Status Bar** ด้านล่างของ VS Code
2. จะเห็นปุ่ม **"No Device Selected"** หรือ **"COM3"** หรือ **"/dev/ttyUSB0"**
3. คลิกที่ปุ่มนั้น
4. เลือก Port ที่ถูกต้อง (ถ้าไม่แน่ใจ ดูใน Device Manager ของ Windows)

```
📍 Windows: มักจะเป็น COM3, COM4, COM5...
📍 macOS: มักจะเป็น /dev/cu.usbserial-XXXX
📍 Linux: มักจะเป็น /dev/ttyUSB0 หรือ /dev/ttyACM0
```

#### ขั้นตอนที่ 2: อัปโหลด

1. คลิกที่ปุ่ม **"→"** (Upload) ใน Status Bar หรือ PlatformIO Toolbar
2. หรือกด `Ctrl + Alt + U`
3. หรือไปที่ **PlatformIO → Upload**

```
┌──────────────────────────────────────────────────────────────┐
│ [✓] Project Tasks                                           │
│    └─ esp32c3 (Platform)                                    │
│       └─ General                                             │
│          └─ [→] Upload                                       │
└──────────────────────────────────────────────────────────────┘
```

#### ขั้นตอนที่ 3: รอการอัปโหลด

1. จะเห็น **Terminal** แสดงข้อความการอัปโหลด:

```
CONFIGURATION: https://docs.platformio.org/page/boards/esp32c3/esp32c3.html
PLATFORM: Espressif 32 (6.4.0)
HARDWARE: ESP32-C3 (40MHz 4MB flash)
RAPID: Programming Size (419844 bytes)
Uploading..
```

2. รอจนเสร็จ (ปกติประมาณ 10-30 วินาที)

#### ขั้นตอนที่ 4: ตรวจสอบว่าสำเร็จ

ถ้าอัปโหลดสำเร็จจะเห็นข้อความ:

```
SUCCESS
Hard resetting via RTS pin...
```

> ⚠️ **ปัญหาที่อาจเจอ:**
> - **"Failed to connect to ESP32-C3"** → ลองกดปุ่ม BOOT ค้างไว้แล้วกด RESET แล้วปล่อย
> - **"No device found"** → ตรวจสอบสาย USB และ Port ที่เลือก
> - **"Permission denied"** → อาจต้องเพิ่มสิทธิ์การใช้งาน Port (Linux/macOS)

### 🔹 เปิด Serial Monitor ดูผลลัพธ์

เมื่ออัปโหลดสำเร็จ บอร์ดจะเริ่มทำงานและส่งข้อความมาที่คอมพิวเตอร์!

#### ขั้นตอนที่ 1: เปิด Serial Monitor

1. ไปที่ **PlatformIO Toolbar** (ด้านล่างของ VS Code)
2. คลิกที่ปุ่ม **"Plug icon"** (Serial Monitor)
3. หรือกด `Ctrl + Shift + M`
4. หรือไปที่ **PlatformIO → Serial Monitor**

#### ขั้นตอนที่ 2: ตั้งค่า Serial Monitor

1. **Baud Rate:** เลือก `115200`
2. จะเห็นข้อความขึ้นมา:

```
========================================
   Hello ESP32-C3! สวัสดีจากบอร์ดของเรา!
========================================
พร้อมรับคำสั่งแล้ว...
บอร์ดกำลังทำงาน... ✅
บอร์ดกำลังทำงาน... ✅
บอร์ดกำลังทำงาน... ✅
...
```

> 💡 **เคล็ดลับ:** ถ้าไม่เห็นข้อความ ลองกดปุ่ม **RESET** บนบอร์ด

## 🔨 ปฏิบัติ: ให้ AI ช่วยแก้ Error

### 🎯 วัตถุประสงค์

ในการทดลองนี้ เราจะได้ฝึกใช้ AI (ChatGPT) ช่วยแก้ปัญหา error ที่อาจเกิดขึ้นขณะเขียนโค้ด

### 🛠️ สิ่งที่ต้องมี

- คอมพิวเตอร์ที่ติดตั้ง VS Code และ PlatformIO แล้ว
- บอร์ด ESP32-C3 (ถ้ามี)
- อินเทอร์เน็ต
- บัญชี ChatGPT

### 📝 ขั้นตอนการทดลอง

#### ขั้นตอนที่ 1: สร้างโค้ดที่มี Error

ลองเขียนโค้ดนี้ใน main.cpp (มี error จงใจ 5 จุด!):

```cpp
// โค้ดนี้มี Error! ลองหาดูสิ
void setup() {
  Serial.begin(115200)
  Serial.println("Hello!");
}

void loop() {
  digitlWrite(LED_BUILTIN, HIGH);  // พิมพ์ผิด!
  delay(1000)
  digitlWrite(LED_BUILTIN, LOW);   // พิมพ์ผิดอีก!
}
```

#### ขั้นตอนที่ 2: สังเกต Error ที่ขึ้น

1. กด **"Build"** (ปุ่มเช็ค ✓) หรือกด `Ctrl + B`
2. ดูที่ **Terminal** ด้านล่าง จะเห็น error สีแดง:

```
.pio/libdeps/esp32c3/ArduinoCore/src/...\main.cpp: In function 'void setup()':
main.cpp:5:20: error: expected ';' before 'Serial'
  Serial.println("Hello!");
                    ^
main.cpp:8:3: error: 'digitlWrite' was not declared in this scope
```

3. **Copy error message** ทั้งหมด (คลิกซ้ายลากเลือก แล้วกด `Ctrl + C`)

#### ขั้นตอนที่ 2: ถาม AI ช่วยแก้

ไปที่ ChatGPT แล้วถามด้วย Prompt นี้:

```
ฉันเขียนโค้ด Arduino สำหรับ ESP32-C3 แล้วเจอ Error เต็มไปหมด 
ช่วยดูให้หน่อยว่าผิดตรงไหน และแก้ให้หน่อย

โค้ดของฉัน:

[วางโค้ดที่มี error ที่นี่]

Error Message:

[วาง error message ที่ copy มาที่นี่]
```

#### ขั้นตอนที่ 3: นำโค้ดที่แก้ไปใช้

1. AI จะตอบมาพร้อมโค้ดที่แก้แล้ว
2. **Copy โค้ดที่แก้แล้ว** มาวางใน main.cpp
3. กด **Build** ใหม่
4. ถ้าไม่มี error แล้ว → อัปโหลดได้เลย!

### 🔍 เทคนิคการถาม AI ให้แก้ Error

#### ✅ วิธีที่ดี:
```
ฉันเขียนโค้ด ESP32-C3 แล้ว error:
"error: 'Serial' was not declared in this scope"

โค้ดฉัน:
void setup() {
  Serial.begin(115200);
}

ช่วยบอกว่าผิดตรงไหน และแก้ให้หน่อย
```

#### ❌ วิธีที่ไม่ดี:
```
Serial มันบอกว่าไม่ declared
```

> 💡 **เคล็ดลับ:** ยิ่งให้ข้อมูลมาก AI ยิ่งช่วยได้ดี! บอกว่าใช้บอร์ดอะไร ใช้ framework อะไร รวมถึง error message ที่เห็น

## 📝 แบบฝึก

### แบบฝึกที่ 1: ติดตั้งโปรแกรม
จงบอกขั้นตอนการติดตั้ง VS Code และ PlatformIO Extension

### แบบฝึกที่ 2: สร้าง Project ใหม่
จงสร้าง Project ชื่อ "MyFirstLED" โดยใช้บอร์ด ESP32-C3

### แบบฝึกที่ 3: แก้ Error จำลอง
จงเปลี่ยนโค้ดนี้ให้ถูกต้อง (มี error 3 จุด):

```cpp
// หาจุดผิด 3 จุด!
void setup()
{
  Serial.begin(115200)  // จุดที่ 1: ลืม ;
  
  for (int i = 0; i < 10; i++)  // จุดที่ 2: ลืม ;
  {
    Serial.println(i)
  }
}

void loop() {
  delay(1000)  // จุดที่ 3: ลืม ;
}
```

<details>
<summary>✅ เฉลย (กดดู)</summary>

```cpp
// โค้ดที่ถูกต้อง:
void setup()
{
  Serial.begin(115200);  // ถูกแล้ว!
  
  for (int i = 0; i < 10; i++) {  // ถูกแล้ว!
    Serial.println(i);  // ถูกแล้ว!
  }
}

void loop() {
  delay(1000);  // ถูกแล้ว!
}
```

จุดที่ 1: ลืม `;` หลัง `115200`  
จุดที่ 2: ลืม `;` หลัง `i < 10`  
จุดที่ 3: ลืม `;` หลัง `1000)` ใน `delay(1000);`  
</details>

### แบบฝึกที่ 4: อธิบายโค้ด
จงอธิบายว่าโค้ดนี้ทำงานยังไง โดยใช้ AI ช่วย:

```cpp
void setup() {
  Serial.begin(9600);
  Serial.println("Counter Started!");
}

void loop() {
  for (int i = 0; i <= 5; i++) {
    Serial.print("Count: ");
    Serial.println(i);
    delay(500);
  }
}
```
