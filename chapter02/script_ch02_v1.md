## 🎬 [7:00-9:00] เขียนโค้ดแรก

### 💻 โค้ด Hello ESP32!

**[On-screen: โค้ด HelloESP32 บน VS Code]**

> 🎙️ **เสียงบรรยาย:**
> "มาเขียนโค้ดแรกกัน! ลบโค้ดเดิมใน main.cpp แล้วพิมพ์ตามนี้..."

```cpp
// บทที่ 2: Hello ESP32 - โค้ดแรกของเรา!
void setup() {
  Serial.begin(115200);  // เริ่มคุยกับคอมพิวเตอร์
  while(!Serial);        // รอจนเชื่อมต่อ
  Serial.println("========================================");
  Serial.println("   Hello ESP32-C3! สวัสดีจากบอร์ดของเรา!");
  Serial.println("========================================");
  Serial.println("พร้อมรับคำสั่งแล้ว...");
}

void loop() {
  Serial.println("บอร์ดกำลังทำงาน... ✅");
  delay(2000);  // รอ 2 วินาที
}
```

> 🎙️ **เสียงบรรยาย:**
> "มาดูโค้ดทีละส่วนนะ..."

**[On-screen: ไฮไลท์โค้ดทีละบรรทัด]**

| คำสั่ง | ทำงานยังไง |
|---------|-------------|
| `Serial.begin(115200)` | เริ่มคุยกับคอมพิวเตอร์ที่ 115200 baud |
| `setup()` | ทำงานแค่ครั้งเดียวตอนเริ่ม |
| `loop()` | ทำงานซ้ำไปเรื่อยๆ |
| `delay(2000)` | หยุดรอ 2 วินาที |

**[On-screen: หน้าจอ error + ChatGPT]**

> 🎙️ **เสียงบรรยาย:**
> "มาฝึกใช้ AI ช่วยแก้ Error กัน!
> ลองเขียนโค้ดนี้ (มี error จงใจ 5 จุด)..."

```cpp
// โค้ดนี้มี Error!
#define LED_PIN 8

void setup() {
  Serial.begin(115200)       // ลืม ; 
  Serial.println("Hello!");
}

void loop() {
  digitlWrite(LED_PIN, HIGH);  // พิมพ์ผิด!
  delay(1000)                  // ลืม ;
  digitlWrite(LED_PIN, LOW);   // พิมพ์ผิด!
}
```

> แล้ววางโค้ด + error message ลงไป
