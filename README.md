## 🛠️ อุปกรณ์ในชุด (~600 บาท)

- ESP32-C3
- DHT11 (เซ็นเซอร์อุณหภูมิ/ความชื้น)
- LDR (เซ็นเซอร์แสง)
- Relay 2CH (ควบคุมไฟ 220V)
- Servo SG90 (มอเตอร์)
- IR Kit (รีโมท)
- Buzzer (เสียง)
- Potentiometer 10K (ปรับค่า)
- Push Button (สวิตช์กด)
- OLED 0.96" I2C (จอ)
- HC-SR04 Ultrasonic (วัดระยะ)
