## 🔨 ปฏิบัติ (พร้อมโค้ด)

### ขั้นตอนที่ 1: ต่อวงจร

[ใส่รูปวงจรที่นี่: `../images/chXX-circuit.png`]

1. ต่อ LED ขาขวาเข้ากับ Resistor 220Ω
2. ต่อ Resistor เข้ากับ GPIO 0 ของ ESP32-C3
3. ต่อ LED ขาซ้ายเข้ากับ GND

### ขั้นตอนที่ 2: เขียนโค้ด

เปิด Arduino IDE แล้วพิมพ์โค้ดนี้:

```cpp
// บทที่ XX: [ชื่อบท]
// ควบคุม LED ให้กระพริบ

#define LED_PIN 0  // กำหนดขา LED = GPIO 0

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);  // ตั้งค่าขา LED เป็นขาออก
  Serial.println("LED Blink Started!");
}

void loop() {
  digitalWrite(LED_PIN, HIGH);  // สั่งให้ LED ติด
  Serial.println("LED ON");
  delay(1000);                  // รอ 1 วินาที

  digitalWrite(LED_PIN, LOW);   // สั่งให้ LED ดับ
  Serial.println("LED OFF");
  delay(1000);                  // รอ 1 วินาที
}
```

### ขั้นตอนที่ 3: อัปโหลดและทดสอบ

1. เลือกบอร์ด **ESP32C3 Dev Module** จาก Tools → Board
2. เลือก Port ที่ถูกต้อง
3. กดปุ่ม **Upload** (ลูกศรชี้ขวา)
4. เปิด **Serial Monitor** ด้วยความเร็ว 115200 baud
5. ดูผลลัพธ์! ✨

> 💡 **เคล็ดลับ:** ถ้าอัปโหลดไม่สำเร็จ ลองกดปุ่ม **BOOT** ค้างไว้แล้วกด **RESET** แล้วปล่อยทีละปุ่ม
