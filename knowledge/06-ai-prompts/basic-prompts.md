## 🌡️ Prompt สำหรับเซ็นเซอร์

### DHT11 (อุณหภูมิ/ความชื้น)

```
"เขียนโค้ด ESP32-C3 อ่านค่าจาก DHT11 แล้วแสดงบน Serial Monitor"
"อุณหภูมิเกิน 35 องศา ให้ buzzer ส่งเสียง"
"บันทึกค่าอุณหภูมิทุก 5 นาที ลง SD card"
```

### LDR (แสง)

```
"เขียนโค้ดวัดความสว่างด้วย LDR แล้วแสดงเป็น %"
"ถ้า LDR ต่ำกว่า 20% ให้เปิดไฟ LED อัตโนมัติ"
"Calibrate LDR ให้ 0% = มืดสนิท, 100% = แสงจ้า"
```

### HC-SR04 (ระยะทาง)

```
"เขียนโค้ดวัดระยะด้วย HC-SR04 แสดงผลเป็น cm"
"ถ้าระยะน้อยกว่า 10cm ให้ Buzzer ส่งเสียงเตือน"
"ทำ parking sensor มี 4 LED แสดงระยะ"
```

### OLED Display

```
"แสดงข้อความ 'Hello' บน OLED 0.96 นิ้ว"
"เขียนโค้ดแสดงอุณหภูมิ + ความชื้นบน OLED พร้อมกัน"
"วาด progress bar บน OLED"
```

## 📝 Prompt Template ที่ดี

### Template 1: บอกบอร์ด + เป้าหมาย + อุปกรณ์

```
"ช่วยเขียนโค้ดสำหรับ [บอร์ด] ให้ [ทำอะไร]
โดยใช้:
- [อุปกรณ์1]
- [อุปกรณ์2]

[GPIO ที่ใช้/แนะนำ]"
```

ตัวอย่าง:
```
"ช่วยเขียนโค้ด ESP32-C3 ให้แสดงอุณหภูมิจาก DHT11 บน OLED
โดยใช้:
- DHT11 (DATA=GPIO 0)
- OLED 0.96" I2C (SDA=0, SCL=1)"
```

### Template 2: มีโค้ดเดิม + ต้องการเพิ่ม

```
"มีโค้ดนี้อยู่แล้ว:
[paste code]

อยากเพิ่ม: [ฟีเจอร์ใหม่]
ช่วยแนะนำว่าต้องเพิ่มอะไร?"
```

### Template 3: มีปัญหา + ต้องการแก้

```
"มีปัญหา: [อาการ]
โค้ดที่ใช้:
[paste code]

แนะนำวิธีแก้หน่อย"
```

## ⚠️ ข้อควรจำ

```
1. AI ก็ผิดได้ - ตรวจสอบก่อนใช้เสมอ
2. บอกบอร์ดให้ชัด - ESP32 ≠ ESP8266 ≠ Arduino
3. ให้โค้ดเดิมด้วย - AI จะตอบได้ตรงกว่า
4. ถามทีละเรื่อง - ถามหลายอย่างพร้อมกันได้คำตอบเละเลย
5. อธิบายว่าทำไม - ไม่ใช่แค่ให้โค้ด แต่ให้เข้าใจด้วย
```
