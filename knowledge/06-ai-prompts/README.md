## หลักการถาม AI ให้ได้คำตอบดี

### ❌ ถามกว้างๆ = ได้คำตอบไม่ดี

```
"ช่วยเขียนโค้ดหุ่นยนต์ให้หน่อย"
"ทำ IoT ยังไง"
"สอนเขียน Arduino"
```

### ✅ ถามเฉพาะเจาะจง = ได้คำตอบดี

```
"เขียนโค้ด ESP32-C3 อ่าน DHT11 แสดงบน OLED"
"อธิบาย PWM signal แบบเข้าใจง่าย"
"โค้ดนี้ผิดตรงไหน: [paste code]"
```

## Prompt Template ที่ดี

### Template 1: บอกบอร์ด + เป้าหมาย + อุปกรณ์

```
"ช่วยเขียนโค้ดสำหรับ [บอร์ด] ให้ [ทำอะไร]
โดยใช้:
- [อุปกรณ์1]
- [อุปกรณ์2]

[GPIO ที่ใช้/แนะนำ]"
```

### Template 2: มีโค้ดเดิม + ต้องการเพิ่ม

```
"มีโค้ดนี้อยู่แล้ว:
[paste code]

อยากเพิ่ม: [ฟีเจอร์ใหม่]
ช่วยแนะนำว่าต้องเพิ่มอะไร?"
```

### Template 3: มีปัญหา + ต้องการแก้

```
"มีปัญหา: [อาการ]
โค้ดที่ใช้:
[paste code]

แนะนำวิธีแก้หน่อย"
```

## Prompt ยอดนิยม

### อธิบายโค้ด

```
"ช่วยอธิบายโค้ดนี้ให้เข้าใจง่ายๆ สำหรับเด็กมัธยมต้น:"
[paste code]
```

### หาบักในโค้ด

```
"มีบักตรงไหนในโค้ดนี้:"
[paste code]
```

### ขอโค้ดตัวอย่าง

```
"เขียนโค้ด ESP32-C3 ให้ LED กระพริบทุก 1 วินาที"
```

### ขอแนะนำ

```
"แนะนำเซ็นเซอร์วัดฝุ่น PM2.5"
"อยากทำ smart home ต้องใช้อะไรบ้าง?"
```

## ⚠️ ข้อควรจำ

```
1. AI ก็ผิดได้ - ตรวจสอบก่อนใช้เสมอ
2. บอกบอร์ดให้ชัด - ESP32 ≠ ESP8266 ≠ Arduino
3. ให้โค้ดเดิมด้วย - AI จะตอบได้ตรงกว่า
4. ถามทีละเรื่อง - ถามหลายอย่างพร้อมกันได้คำตอบเละเลย
5. อธิบายว่าทำไม - ไม่ใช่แค่ให้โค้ด แต่ให้เข้าใจด้วย
```

## 🎯 ตัวอย่าง Prompt ที่ดีที่สุด

```
"ช่วยอธิบายโค้ดนี้ทีละบรรทัด แล้วบอกว่า
มันทำงานยังไง: [paste code]"

"ช่วยเขียนโค้ด ESP32-C3 เปิดพัดลมอัตโนมัติ 
เมื่ออุณหภูมิเกิน 30°C โดยใช้ DHT11 + Relay"

"มีโค้ดนี้ [paste code] แล้วมีปัญหาว่า 
LED ไม่ติด ช่วยหาสาเหตุหน่อย"
```
