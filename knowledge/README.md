# 📖 ESP32-C3 STEM Knowledge Base

> คลังความรู้สำหรับการเรียนรู้ ESP32-C3 และ IoT พื้นฐาน

## 📚 โครงสร้างความรู้

```
knowledge/
├── 01-fundamentals/     🔰 พื้นฐานอิเล็กทรอนิกส์
├── 02-sensors/          🌡️ เซ็นเซอร์ต่างๆ
├── 03-actuators/         ⚡ อุปกรณ์ขับเคลื่อน
├── 04-communication/      📡 การสื่อสาร
├── 05-projects/          🏗️ โปรเจกต์ตัวอย่าง
├── 06-ai-prompts/       🤖 Prompt สำหรับ AI
├── 07-troubleshooting/   🔧 แก้ปัญหาที่พบบ่อย
└── 08-teaching/         👨‍🏫 แนะนำการสอน
```

## 🎯 วิธีใช้งาน

1. **เริ่มต้น** → อ่าน `01-fundamentals/` ก่อน
2. **เลือกหัวข้อ** → ไปที่หมวดที่สนใจ
3. **ถาม AI** → ใช้ Prompt ใน `06-ai-prompts/`
4. **แก้ปัญหา** → ดูใน `07-troubleshooting/`

## 🚀 เริ่มต้นเร็ว

| หัวข้อ | ไฟล์ | ระดับ |
|---------|-------|--------|
| ESP32-C3 คืออะไร | `01-fundamentals/esp32c3-basics.md` | 🔰 ผู้เริ่มต้น |
| GPIO การใช้งาน | `01-fundamentals/gpio-guide.md` | 🔰 ผู้เริ่มต้น |
| วงจรพื้นฐาน | `01-fundamentals/circuit-basics.md` | 🔰 ผู้เริ่มต้น |

## 🤖 การใช้ AI ช่วย

```
1. ถาม AI ตาม Prompt ในโฟลเดอร์ 06-ai-prompts/
2. อ่านคำอธิบาย + ดูโค้ดตัวอย่าง
3. ทดลองทำตาม
4. ถาม AI เมื่อติดปัญหา
```
